2. Keyboard Layout Dedign

Keyboard Layout Editorを使ってKeyboardのレイアウトを行う

こちらによると Keyboard Layout Editorを使って配置を決めるそのjsonデータをKiCadにインポートすることができる。まずはここの記事を参考にして Keyboard Layout Editorでキーボードのレイアウトを作成する。キーの位置はもちろん、回転や大きさやENTERのような複雑な形状も 作成が可能。キーボードの形はrev3を参考にして ハ型・分裂型のキーボードを作成する。

tools

  • https://www.keyboard-layout-editor.com
  • https://github.com/Salicylic-acid3/PCB_Data

タスク

  • キーボードの配置を決める
  • キーボードの配置をKeyboard Layout Editorを使って実装する
  • KiCadを使って配線と回路を決定する。

セパレートの結合

  • UARTかI2Cを使って通信する.
  • raspberry pi pico Wを使ってBluetooth接続をする
